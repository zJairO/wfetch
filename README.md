# wfetch

![](https://repository-images.githubusercontent.com/428005882/dfae335c-e8e6-4af0-9360-5a15b6e86978)

wfetch is a Minimal Windows system information tool written in Python 

## Installation
```bash
pip install wfetch
```
## Usage
```bash
python -m wfetch
```
